# CLAUDE.md

Family household management platform with calendar sync, chore tracking, and wall-mounted Raspberry Pi displays.

## Commands

```bash
npm run dev          # Dev server at localhost:3000
npm run build        # Production build
npm run typecheck    # TypeScript check
npm run lint         # ESLint
npm run test         # Vitest
```
